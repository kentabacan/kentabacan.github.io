My Projects

SnackChat

SnackChat is our take on a restaurant review service. This webapp was built using JavaScript, MongoDB, and PHP. It uses a variety of data streams, such as current location and check-in times to develop personalized models and produce tailored search results. It was built for a university class project in a group of four. View the code here and here.

See it in action

Starship Enterprise HUD

The Starship Enterprise HUD is an AR (augmented reality) proof of concept heads-up-display built in Unity. It lets users navigate a futuristic UI to search for places of interest, such as dining areas. We implemented hand gestures and voice commands using LeapMotion and the FourSquare API. It was developed in under 48 hours for LA Hacks 2018 in a team of four. View the code here.

See it in action

Pokegatchi

Pokegatchi is an Android video game utilizing AR, taking elements of both Pokémon and Tamagotchi. Users are able to convert physical Pokémon cards into a virtual environment for a variety of minigames. The game was created with Unity and Vuforia. It was developed in under 36 hours for HackUCI 2018 in a team of five. It won "Most Addicting App" and was a runner-up for "Most Polished Game". View the code here.

Maverick

Maverick is a video game built with Unity. In it, players navigate the protagonist through a dangerous highway in an action-packed shootout. It was developed in under 36 hours for Game Maker's Toolkit Game Jam 2017 in a team of four. View the code here.
